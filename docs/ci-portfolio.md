The portoflio is a summary of 6 repos. The 6 repos taught the full workflow of building a real monitoring pipeline: first setting up GitHub, VS Code, `uv`, and Python project structure, then learning how to detect static anomalies, design useful signals from raw data, monitor behavior over time with rolling windows, detect drift when a system changes from its baseline, and finally combine everything into one complete pipeline that reads data, creates signals, detects problems, saves artifacts, and summarizes system health for decision-making.
